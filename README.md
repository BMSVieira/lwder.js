◼️ Installation:
-

<b>1 - Include JavaScript Source.</b>
```javascript
<script src="path/to/lwder.js"></script>
```
<b>2 - Set HTML.</b>
```html
<div class="lwder" lwder="..." lwderspeed="300"> Page is loading </div>
```
<b>3 - Initilize.</b>
```javascript
document.addEventListener("DOMContentLoaded", function() {
 new Lwder({ selector: ".lwder" });
});
```
